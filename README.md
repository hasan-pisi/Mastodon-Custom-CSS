MAVZERON: Atomic Silver & Liquid Mercury Ecosystem

Mavzeron is an advanced CSS ecosystem that deconstructs the standard Mastodon interface and replaces it with a futuristic command center built from brushed steel, liquid mercury, and platinum glass textures. This project merges the industrial "Mavzer" aesthetic with the fluidity of the digital world.
💎 Core Features

    Atomic Silver Interactions: Favorite, Boost, and Reply icons are purged of their default colors (purple, red, blue). They glow like a silver mist on hover and lock into a polished chrome finish when activated.

    Platinum Glass Panels: All columns and administrative panels feature a "Frosted Glass" design with 90% opacity and 25px blur, creating a floating sense of depth.

    Mercury Logic Switches: The standard Mastodon toggle switches are replaced with silver gradients, providing a liquid mercury effect when toggled.

    Chrome Reflect Logo: A dynamic "MAVZERON" header featuring an animated reflection effect, heavy font-weight (900), and wide letter-spacing.

    Industrial Timeline: Posts (Slabs) feature a "Silver Rail" effect; they glow and slide slightly to the right upon hovering, emphasizing the active content.

    Unified Header & Filters: The "Home" header and filter settings are fully integrated into the silver armor, ensuring a seamless visual flow across the middle column.

🚀 Installation
1. For Instance Administrators (Instance-Wide)

If you run a Mastodon instance and want to provide this experience to all users:

    Navigate to Administration > Settings > Appearance.

    Paste the entire code from this repository into the Custom CSS box.

    Save changes and initiate the silver revolution.

2. Personal Use (via Stylus)

To use Mavzeron only for your own account:

    Install the Stylus extension (Chrome/Firefox).

    Create a new style for your Mastodon domain.

    Paste the CSS code and activate it.

🎨 Design Philosophy

Mavzeron aims to connect the user with the noble, cold texture of industrial software. The radial-gradient and cubic-bezier transitions used in the design are mathematically calculated to simulate real-world metallic reflections. To reduce visual clutter, all icons are pulled into a monochrome hierarchy; "energy" (glow) is only released during interaction.
🛠️ Technical Specifications

    Typography: Montserrat (Sans-serif)

    Accent Colors: #ffffff (Chrome Bright), #c0c0c0 (Silver), #1a1c1e (Dark Metal)

    Key Effects: Backdrop-filter (Blur), Drop-shadow (Glow), Keyframe Animations.
